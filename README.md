# Парсер таблицы сотрудников

Утилита командной строки для чтения, парсинга и красивого вывода таблицы со списком сотрудников. Программа поддерживает чтение данных как из локальных файлов, так и с веб-сервера.

## Описание

Скрипт обрабатывает текстовый файл с информацией о сотрудниках, содержащей:
- **ФИО** - Фамилия, имя, отчество
- **Возраст** - Численное значение возраста
- **Адрес** - Адрес проживания (улица, номер дома, город)
- **Дата рождения** - В различных форматах

### Особенности

✅ **Поддерживает множество форматов дат:**
- `YYYY-MM-DD HH:MM:SS`
- `YYYYMMDD HH:MM:SS`
- `YYYYMMDDTHHMMSS`
- `YYYYMMDD HHMMSS`

✅ **Интеллектуальный парсинг адресов:**
- Замена сокращений (ул. → улица, пр. → проспект, пер. → переулок и т.д.)
- Автоматическая перестановка города и улицы при необходимости
- Корректная обработка адресов с переменным количеством табуляций

✅ **Гибкий ввод данных:**
- Чтение из локальных файлов
- Чтение с веб-сервера (HTTP/HTTPS)
- Кодировка windows-1251 (cp1251)

✅ **Красивый вывод:**
- ASCII таблица с автоматической подстройкой под размер терминала
- Поддержка русского языка
- UTF-8 кодировка в консоли

## Требования

- Python 3.7+
- Библиотека `urllib` (входит в стандартную библиотеку Python)

## Использование

### Базовое использование

```bash
python script.py --input <путь_к_файлу_или_url>
```

### Примеры

**Чтение из локального файла:**
```bash

python script.py --input /path/to/data.txt
```

**Чтение с веб-сервера:**
```bash
python script.py -i https://example.com/data.txt
```

### Параметры

| Параметр | Описание |
|----------|---------|
| `-i, --input` | Обязательный. Путь к файлу или URL для чтения данных |

## Формат входного файла

Файл должен быть в формате TSV (табуляция как разделитель) с кодировкой windows-1251:

```
ФИО                     [TAB] Возраст [TAB] Адрес                              [TAB] Дата рождения
Иванов Иван Иванович    [TAB] 30      [TAB] улица Пушкина, 42, Москва        [TAB] 1993-12-14 05:44:30
Петров Петр Петрович    [TAB] 45      [TAB] ул. Советская, 10, Казань        [TAB] 1978-06-20 15:30:45
```

## Описание функций

### `read_source(source: str) -> bytes`
Читает данные из файла или с веб-сервера.

**Параметры:**
- `source` - Путь к локальному файлу или URL

**Возвращает:**
- Байты с содержимым файла

### `parse_date(raw: str) -> str`
Парсит дату из строки в различных форматах и возвращает в стандартизированном формате `YYYY-MM-DD HH:MM:SS`.

**Параметры:**
- `raw` - Строка с датой в различных форматах

**Возвращает:**
- Дата в формате `YYYY-MM-DD HH:MM:SS` или исходная строка если парсинг не удался

### `normalize_address(addr: str) -> str`
Нормализует адрес: заменяет сокращения и переставляет город и улицу при необходимости.

**Параметры:**
- `addr` - Адрес для нормализации

**Возвращает:**
- Нормализованный адрес

### `parse_lines(text: str) -> List[List[str]]`
Парсит текст и выделяет информацию о сотрудниках в структурированный формат.

**Параметры:**
- `text` - Текст с данными сотрудников

**Возвращает:**
- Список списков [ФИО, Возраст, Адрес, Дата рождения]

### `render_table(rows: List[List[str]], title: str = 'Список сотрудников') -> None`
Выводит данные в виде красивой ASCII таблицы.

**Параметры:**
- `rows` - Список строк таблицы
- `title` - Заголовок таблицы (по умолчанию "Список сотрудников")

## Обработка ошибок

Программа обрабатывает следующие ошибки:

- **Файл не найден** - Выводит ошибку при чтении локального файла
- **Сетевая ошибка** - Выводит ошибку при чтении с веб-сервера
- **Пустые данные** - Выводит сообщение "Данные не найдены"
- **Ошибки кодировки** - Использует `errors='replace'` для обработки некорректных символов

## Известные ограничения

1. Возраст должен быть в диапазоне 1-150 лет
2. Город должен быть в списке известных городов для перестановки (Москва, Санкт-Петербург, Екатеринбург, Новосибирск, Казань)
3. Максимальный размер файла ограничен доступной памятью
4. Поддерживается только кодировка windows-1251 (cp1251)
