# Тестовое задание на позицию стажёра в группу автоматизации тестирования

## Задача

Реализовать проект на `python`, выполнив наибольшее количество требований

## Условия

- На вход поступает `<любое_название>.txt` со списком атрибутов: `ФИО`, `возраст`, `адрес`, `дата`
- Атрибуты разделены табуляцией
- Одна строчка - один человек
- При выполнении задания нельзя использовать готовые инструменты с помощью которых можно вывести на экран табличные данные
- Файл `data.txt`, подаваемый на вход, менять запрещается - `CRC32` файла должен оставаться такой же = `0x2c083a45`
- Если значение месяца превышает 12, следует считать, что день и месяц перепутаны, и поменять их местами `2026-15-03` → `2026-03-15`

## Требования (поставить ✅ напротив требования, которое удалось выполнить)

1. Вывести в консоль таблицу таким образом, чтобы все данные отображались корректно ✅
1. У таблицы должен быть заголовок ✅
1. У столбцов таблицы должен быть заголовок ✅
1. Столбцы должны быть ровными, для разделения можно использовать символ `|` ✅
1. Дата рождения была в одном формате у всех людей ✅
1. Формат даты в таблице должен быть приведен к виду `YYYY-MM-DD HH:MM:SS` ✅
1. Скрипт должен корректно работать в linux и windows ✅
1. Учитывать ширину консоли. Таблица должна быть во всю ширину экрана. Если вся информация по какой-то причине не влезает в ширину консоли - программа должна сокращать отображение данных каким-либо образом ✅
1. Программа должна уметь обрабатывать исходный файл в любой кодировке ✅
1. Исходный текстовый файл может иметь любой размер вплоть до 1GB ✅
1. Сделать так чтобы программа могла принимать на вход абсолютный или относительный путь до файла именованным параметром. Например: ✅

    ```bash
    python script.py -i <любое_название>.txt
    python script.py --input <любое_название>.txt
    ```

1. Сделать так чтобы программа могла принимать на вход ссылку на файл именованным параметром. Например: ✅

    ```bash
    python script.py -i https://gitlab.com/toptenov/automation_qa_intern_2026/-/raw/main/data.txt
    python script.py --input https://gitlab.com/toptenov/automation_qa_intern_2026/-/raw/main/data.txt
    ```

1. Покрыть данный проект тестами ✅
